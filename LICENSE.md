## 🛠️ Troubleshooting Common Issues

**❌ Fluxus Z Executor fails to inject after Roblox update**
Sometimes a new Roblox client build changes memory offsets that the injection engine relies on.
**Solution:** Check for a Fluxus Z Executor patch — updates typically ship within 24 hours of a Roblox engine change. Restart both applications after updating.

**❌ Antivirus quarantines the installer file**
Some security software flags script execution tools with heuristic detections regardless of actual behavior.
**Solution:** Add the Fluxus Z Executor install directory (e.g., `C:\Program Files\FluxusZ\`) to your antivirus exclusion list before installing.

**❌ Fluxus Z Executor Apk crashes on launch (Android)**
This usually occurs on devices with under 3 GB of free RAM or outdated GPU drivers.
**Solution:** Close background applications, ensure your Android OS is updated, and reinstall the latest Fluxus Z Executor Apk build.

**❌ Scripts execute but produce no visible effect**
The script may depend on API functions that the current Roblox experience does not expose.
**Solution:** Verify script compatibility in the Fluxus Z Executor script hub notes. Try running a simple test script like `print("hello")` to confirm the engine is functioning.

**❌ Cloud sync shows conflict errors**
Simultaneous edits from desktop and mobile can create version conflicts in cloud storage.
**Solution:** Open the sync panel in Fluxus Z Executor settings and manually select which version to keep. Enable the `auto-resolve newest` option to prevent future conflicts.

**❌ Editor is laggy when handling large scripts**
Scripts exceeding 5,000 lines can strain the syntax highlighter in real time.
**Solution:** Disable `highlight_active_line` in the Fluxus Z Executor configuration file or split large scripts across multiple tabs.
